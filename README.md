**Accounting Information System**
* Project Overview

This project is an Accounting Information System developed to manage and monitor financial transactions efficiently. The system provides features for inputting transactions, managing the chart of accounts (COA), and generating financial reports such as Ledgers, Balance Sheets, and Income Statements.

* Purpose

The main goal of this project is to create a digital platform that helps users record financial transactions and automatically generate financial reports in a structured and efficient way.

* Tools & Technologies

Programming Language: PHP

Database: MySQL (via phpMyAdmin)

Frontend: HTML, CSS (basic)

Local Server: XAMPP

* Main Features

Journal Management: Input and view transaction data

Chart of Accounts (COA): Add and manage account lists

* Financial Reports:

Ledger

Balance Sheet

Income Statement

* System Structure

journal/ – Transaction input and transaction list

coa/ – Chart of Accounts input and list

report/ – Financial report pages (ledger, balance sheet, income statement)

* Outcome

The project successfully simulates how an accounting information system processes and organizes financial data. It can be used as a basic prototype for small business accounting systems or as a learning project in accounting information systems courses.
